# flag-raising-game
旗上げゲームをhtmlで作成するレポジトリです

## 実装方針
このレポジトリはREADEMEとindex.htmlの二つで構成されます。
READMEが実装するアプリケーションの説明index.htmlがアプリの本体です。

## 実装内容
単純な旗上げゲームを実装します。
ルールは5秒に一回赤あげて、赤下げてなど指示をだし、それに従うゲームです。
操作は旗を上げるボタンを押下することで操作します。
一度でも失敗したらゲームオーバーです。
30回連続で成功したらクリアです。

5秒間の間は何度でも上げ下げできます。
5秒がった時のステータスで成功可否を判定してください。
